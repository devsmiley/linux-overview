# VIM Editor

```
l = move right
k = move up
j = move down
h = move left
x = delete letter
u = undo the change
Ctrl+r = redo the change
r <new_char> = replaces letter with new_char
dd = delete a line
<no.> dd = deletes no of lines specified
i = enter insert mode
v = enter visual mode
y = copy the text
p = paste the text after cursor
P = paste the text before cursor
d = cut the text
```
